\section{Preliminaries}
\label{sec:preliminaries}

\subsection{Notation}

All vector spaces are finite-dimensional over $\R$. Inner products are written $\ip{x}{y}$. Vector lengths are written with explicit subscripts; in this paper they use the $\ell_2$ norm $\norm{x}_2$. For matrices, $\norm{A}_F$ denotes the Frobenius norm, $\ker A$ denotes the nullspace, and $\Span(A)$ denotes the span of the columns of $A$.

For a matrix $K$, the Moore-Penrose pseudoinverse is denoted by $K^\dagger$. The matrix $KK^\dagger$ is the orthogonal projector onto $\Span(K)$. If $K=[k_1,\ldots,k_T]\in\R^{r\times T}$ and $V=[v_1,\ldots,v_T]\in\R^{m\times T}$, the equation $WK=V$ means that the memory $W\in\R^{m\times r}$ binds each key $k_t$ to the corresponding value $v_t$.

\subsection{Projection geometry}

Let $S$ be a closed convex subset of a finite-dimensional real vector space. Let $\psi:S\to\R$ be differentiable and strictly convex. The Bregman divergence generated by $\psi$ is
\begin{align*}
D_\psi(u,v)=\psi(u)-\psi(v)-\ip{\nabla\psi(v)}{u-v}.
\end{align*}
When $\psi(s)=\norm{s}_2^2/2$, this is squared $\ell_2$ distance. All projection statements below are finite-dimensional and use the standard variational characterization of convex minimizers.

\subsection{Associative evidence}

A memory state is a matrix $W\in\R^{m\times r}$. A collection of key-value evidence is written as
\begin{align*}
K=[k_1,\ldots,k_T]\in\R^{r\times T},\qquad V=[v_1,\ldots,v_T]\in\R^{m\times T}.
\end{align*}
Exact binding means $WK=V$. A new key $k\in\R^r$ is decomposed relative to the old key span by
\begin{align}\label{eq:innovation}
\Pi=KK^\dagger,\qquad z=(I-\Pi)k,\qquad a=K^\dagger k.
\end{align}
The vector $z$ is the innovation: the component of $k$ not explained by previous keys. The coefficient vector $a$ gives the least-squares representation of the old-span component of $k$.

\subsection{Statement convention}

The paper uses two levels of theorem statements. Main-text theorems are informal versions: they are mathematically precise enough to communicate the result and proof method, but omit some local regularity and typing details when those are fixed in the surrounding text. Each main-text theorem points to a formal appendix theorem. Each formal appendix theorem links back to its main-text version.

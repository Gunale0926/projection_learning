\section{Information Structures and Missing Distinctions}
\label{sec:preliminaries}

The question ``should the learner repair or refine?'' is a question about indistinguishability. We use partitions and sigma-algebras to make that question precise. In the finite case, an atom of a partition is a set of worlds that the learner currently treats as identical. In the general case, a sigma-algebra records exactly which events are observable. A predictor can use only distinctions present in this information structure.

\subsection{Information as distinctions}

In the finite setting, an information structure is a partition
\begin{align*}
\calP=\{B_1,\ldots,B_n\}
\end{align*}
of $\Omega$ into nonempty disjoint atoms. A partition $\calQ$ refines $\calP$ if every atom of $\calQ$ is contained in an atom of $\calP$. An event $E\subseteq\Omega$ is $\calP$-measurable if it is a union of atoms of $\calP$. Thus a partition is not only notation. It is the learner's current quotient of the world into distinguishable cases.

For sigma-algebras $\calF,\calG\subseteq\calA$, the inclusion $\calF\subseteq\calG$ means that $\calG$ is at least as informative as $\calF$. The join $\calF\vee\calG$ is the smallest sigma-algebra containing both. For a random variable $Z$, $\sigma(Z)$ is the sigma-algebra generated by $Z$; for an event $E$, $\sigma(E)=\{\varnothing,E,E^c,\Omega\}$.

\subsection{Measurable predictors}

A measurable predictor is what the learner is allowed to compute from its current distinctions. In a finite partition, a prediction or action $f:\Omega\to A$ is $\calP$-measurable if it is constant on each atom:
\begin{align*}
\omega,\omega'\in B\in\calP\quad \Longrightarrow\quad f(\omega)=f(\omega').
\end{align*}
Equivalently, $f$ can depend only on which atom contains the current world state. In the sigma-algebra setting, admissible predictors are $\calF$-measurable functions. This is the formal reason a missing distinction cannot be repaired by a more clever local update inside the same atom: the update still returns a function on the same quotient.

\subsection{Bayes risk under information}

Bayes risk under information is the best possible performance given the distinctions currently available. Let $Y:\Omega\to\calY$ be a target random variable, let $A$ be an action space, and let $L:\calY\times A\to\R$ be a loss for which the expectations below are well-defined. The Bayes risk available under information $\calF$ is
\begin{align}
R_L(\calF)=\inf\{\E[L(Y,a)]:a:\Omega\to A,\ a\text{ is }\calF\text{-measurable}\}.
\label{eq:bayes-risk}
\end{align}
Thus $R_L(\calF)$ is the best population risk achievable by a predictor that can use only distinctions in $\calF$, following the standard decision-theoretic view of actions, losses, Bayes risk, and comparison of information structures \citep{blackwell1953equivalent,berger1985statistical}.

\subsection{Contradiction inside an atom}

The central obstruction is immediate. If an atom contains states that require different outputs, the current information structure does not merely have a bad parameter value; it lacks the expressive degree of freedom needed to represent the target.

\begin{theorem}[Contradiction requires distinction]
\label{thm:contradiction:informal}
Let $\calP$ be a partition of $\Omega$ and let $y:\Omega\to A$ be a required output. Suppose $\omega,\omega'$ lie in the same atom of $\calP$ but $y(\omega)\ne y(\omega')$. Then no $\calP$-measurable predictor $f:\Omega\to A$ satisfies both $f(\omega)=y(\omega)$ and $f(\omega')=y(\omega')$.
\end{theorem}

This theorem is deliberately architecture-independent. It applies to a neural cache, a local adapter, a fast-weight memory, a nearest-neighbor system, or a test-time optimizer whenever the update is constrained to the current information. If the relevant states are not separated, no repair rule inside the atom can represent the required behavior. The contradiction is evidence that the atom should not exist.

\subsection{Notation used later}

All probability spaces are denoted by $(\Omega,\calA,\mathbb P)$ unless the finite case is stated explicitly. Random variables are functions on $\Omega$. We use standard measure-theoretic probability notation for generated sigma-algebras, joins, measurability, and conditional expectation \citep{billingsley1995probability,kallenberg2002foundations}. For vectors $x\in\R^d$, lengths use the $\ell_2$ norm $\norm{x}_2$. For matrices, $\norm{M}_F$ is the Frobenius norm. Inner products are written $\ip{x}{y}$. For a matrix $K$, the Moore-Penrose pseudoinverse is $K^\dagger$, and $KK^\dagger$ is the orthogonal projector onto the span of the columns of $K$ \citep{penrose1955generalized}.

For the bridge to Projection Memory, we use a finite partition $\calP=\{B_1,\ldots,B_n\}$ and its one-hot feature map $\phi_\calP:\Omega\to\R^n$:
\begin{align*}
\phi_\calP(\omega)=e_i\qquad \text{if}\qquad \omega\in B_i.
\end{align*}
A matrix $W\in\R^{m\times n}$ assigns a value vector to each atom through $W\phi_\calP(\omega)$. For old key matrices $K$, a new key $k$ has CPM innovation
\begin{align}
z=(I-KK^\dagger)k.
\label{eq:cpm-innovation}
\end{align}

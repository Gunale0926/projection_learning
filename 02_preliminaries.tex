\section{Preliminaries}
\label{sec:preliminaries}

\subsection{Notation}

All probability spaces are denoted by $(\Omega,\calA,\mathbb P)$ unless the finite case is stated explicitly. Random variables are functions on $\Omega$. We use standard measure-theoretic probability notation for sigma-algebras, generated sigma-algebras, joins, measurability, and conditional expectation \citep{billingsley1995probability,kallenberg2002foundations}. For vectors $x\in\R^d$, lengths use the $\ell_2$ norm $\norm{x}_2$. For matrices, $\norm{M}_F$ is the Frobenius norm. Inner products are written $\ip{x}{y}$. For a matrix $K$, the Moore-Penrose pseudoinverse is $K^\dagger$, and $KK^\dagger$ is the orthogonal projector onto the span of the columns of $K$ \citep{penrose1955generalized}.

For sigma-algebras $\calF,\calG\subseteq\calA$, the inclusion $\calF\subseteq\calG$ means that $\calG$ is at least as informative as $\calF$. The join $\calF\vee\calG$ is the smallest sigma-algebra containing both. For a random variable $Z$, $\sigma(Z)$ is the sigma-algebra generated by $Z$; for an event $E$, $\sigma(E)=\{\varnothing,E,E^c,\Omega\}$.

\subsection{Partitions and measurable predictors}

In the finite setting, an information structure is a partition
\begin{align*}
\calP=\{B_1,\ldots,B_n\}
\end{align*}
of $\Omega$ into nonempty disjoint atoms. A partition $\calQ$ refines $\calP$ if every atom of $\calQ$ is contained in an atom of $\calP$. An event $E\subseteq\Omega$ is $\calP$-measurable if it is a union of atoms of $\calP$.

A prediction or action $f:\Omega\to A$ is $\calP$-measurable if it is constant on each atom:
\begin{align*}
\omega,\omega'\in B\in\calP\quad \Longrightarrow\quad f(\omega)=f(\omega').
\end{align*}
Equivalently, $f$ can depend only on which atom contains the current world state. In the sigma-algebra setting, admissible predictors are $\calF$-measurable functions.

\subsection{Bayes risk under information}

Let $Y:\Omega\to\calY$ be a target random variable, let $A$ be an action space, and let $L:\calY\times A\to\R$ be a loss for which the expectations below are well-defined. The Bayes risk available under information $\calF$ is
\begin{align}
R_L(\calF)=\inf\{\E[L(Y,a)]:a:\Omega\to A,\ a\text{ is }\calF\text{-measurable}\}.
\label{eq:bayes-risk}
\end{align}
Thus $R_L(\calF)$ is the best population risk achievable by a predictor that can use only distinctions in $\calF$, following the standard decision-theoretic view of actions, losses, Bayes risk, and comparison of information structures \citep{blackwell1953equivalent,berger1985statistical}.

\subsection{Projection memory notation}

For the bridge to Projection Learning, we use a finite partition $\calP=\{B_1,\ldots,B_n\}$ and its one-hot feature map $\phi_\calP:\Omega\to\R^n$:
\begin{align*}
\phi_\calP(\omega)=e_i\qquad \text{if}\qquad \omega\in B_i.
\end{align*}
A matrix $W\in\R^{m\times n}$ assigns a value vector to each atom through $W\phi_\calP(\omega)$. For old key matrices $K$, a new key $k$ has CPM innovation
\begin{align}
z=(I-KK^\dagger)k.
\label{eq:cpm-innovation}
\end{align}

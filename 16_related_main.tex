\section{Related Work}
\label{sec:main-related}

\paragraph{Efficient sequence mixers and expressive state.}
Transformers use content-addressed attention over previous tokens
\citep{vaswani2017attention}.
Mamba, RWKV, GDN, GLA, RetNet, and related models replace attention with
efficient recurrent or linear state updates
\citep{gu2023mamba,peng2023rwkv,yang2025gated,yang2024gated,sun2023retentive}.
TTT-style models instead make the hidden state expressive by running a learned
update rule at test time \citep{sun2025learning}.
Our experiments place CGA in this comparison class: CGA is an attention-like
causal mixer whose persistent state is a bounded concept memory rather than a
token table.

\paragraph{Attention variants.}
Sparse attention, sliding-window attention, grouped-query attention, and
compressed-memory attention mainly alter which token keys and values are read
or how the token table is stored.
CGA is orthogonal: it changes the basis being attended to.
The read still has the same softmax form, so optimized attention kernels and
attention variants can in principle be adapted to concept slots instead of
token positions.

\paragraph{Associative and external memory.}
Cache language models, $k$NN-LMs, retrieval-augmented systems, fast weights,
online ridge or RLS memories, Memory Networks, Neural Turing Machines, and
Differentiable Neural Computers all use forms of stateful retrieval or local
memory repair
\citep{grave2017improving,khandelwal2020generalization,lewis2020retrieval,ba2016using,schlag2021linear,crammer2006online,weston2015memory,graves2014neural,graves2016hybrid}.
CGA shares the associative-memory goal but keeps the decoder-block interface of
attention: each token reads from concept slots and writes back through a gated
causal update.

\paragraph{Routing and capacity.}
Mixture-of-experts, Switch Transformers, and Product-Key Memory route tokens or
queries to trained capacity
\citep{shazeer2017outrageously,fedus2022switch,lample2019large}.
CGA also routes writes, but the routed capacity is runtime state rather than a
bank of trained expert parameters.
The fixed concept count $C$ makes this capacity explicit and state-equalizable
against recurrent mixers.

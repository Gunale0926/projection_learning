Self-attention stores one key-value entry per previous token and retrieves from
that token table. This gives expressive content-addressed recall, but it ties
the memory basis to token events and makes the read state grow with context
length. We propose \emph{Concept Graph Attention} (CGA), a causal sequence
layer that keeps the continuous readout of attention but replaces token memory
with a fixed bank of concept identities. Each concept has a fixed address and a
runtime value memory. Tokens read from the concept bank with attention-style
softmax weights and write back to the same bank with gated continuous updates
and trainable retention.

CGA therefore changes where attention is computed: from token positions to a
bounded concept memory whose contents evolve over the prefix. In the current
formulation, the online state is only the concept value table, so the state size
is $C H d_v^c$ for $C$ concepts, $H$ heads, and per-head concept value width
$d_v^c$. During training, gradients flow through the concept read, the
continuous write, the write gate, and the retention parameters. During
inference, model weights remain fixed, but the concept memory is updated by the
same forward rule from the observed prefix. This gives CGA test-time learning
of memory contents without an inference-time backward pass.

Empirically, we organize the evaluation around two ML-community baselines:
state-equalized multi-query associative recall in the PoST/Zoology framework,
and matched language-model pretraining on FineWeb-Edu at 180M--440M scale with
standard zero-shot LM Harness evaluation. The intended comparison class is
modern efficient attention alternatives such as Mamba, RWKV, Gated DeltaNet,
and TTT-style stateful layers. CGA keeps attention-like retrieval, but moves
the persistent state from token memory or learned test-time weights to a
bounded, updateable concept memory.

We formalize online prediction with refinable information structures. At each time, a learner's runtime state induces a partition $\Pi_t$ of a finite case space: repair changes the predictor on the current atoms, while refinement changes the partition itself by adding a binary distinction. We prove a repair-only lower bound: for any fixed partition $\Pi$, every learner whose predictions remain $\Pi$-measurable suffers expected regret at least $T(R_L(\Pi)-R_L(\Pi^\star))$ relative to a refined oracle $\Pi^\star$. Thus some runtime errors are not optimization errors but quotient errors. We characterize useful refinements by Bayes actions: for $\calF\subseteq\calG$, the refinement gap vanishes iff the $\calG$-Bayes action is already $\calF$-measurable. Under log loss this gap is $\E D_{\mathrm{KL}}(P(Y\mid\calG)\|P(Y\mid\calF))$, and a binary distinction has value $I(Y;U\mid\calF)$. We give finite-sample validation rules, distinction-complexity parameters, and realizable recovery guarantees for greedy refinement over finite distinction classes. Finally, we instantiate the theory as Distinction Graph Machines, a dynamic graph data structure whose nodes repair local memories and whose edges generate accepted distinctions. Controlled streams serve as separating examples: repair-only learners fail on missing-distinction tasks, while refine-and-repair learners recover the needed partition.

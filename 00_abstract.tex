We prove a repair obstruction for online prediction.
For any fixed information partition $\Pi$, every adaptive learner whose predictions remain $\Pi$-measurable has expected regret at least
\[
T\left(R_L(\Pi)-R_L(\Pi^\star)\right)
\]
against a refined oracle partition $\Pi^\star$, up to the standard epsilon-oracle relaxation when Bayes actions are not attained.
Thus a positive refinement gap is an information-structure error: no update that preserves $\Pi$ can remove it.

We characterize useful refinements.
Adding a prediction-time binary distinction has zero value exactly when the refined Bayes action is already measurable under the old information; under log loss its value is $I(Y;U\mid\calF)$, and under squared loss it is conditional variance reduction.

We instantiate the escape operation as Distinction Graph Machines, a dynamic graph-memory data structure whose edges are accepted distinctions and whose nodes store repairable local memories.
Controlled streams separate repair-only learners from refine-and-repair learners.

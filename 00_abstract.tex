Can a runtime learner fix an error without changing what it can condition on? We formalize this as a problem over information structures. At time $t$ the learner has a partition, or sigma-algebra, $\calF_t$; repair changes only an $\calF_t$-measurable predictor or local state, while refinement enlarges $\calF_t$. We prove an architecture-independent obstruction: if two states in one atom require different consequences, every repair constrained to the current information fails. We then prove the canonical update theorem: the least refinement that makes new evidence $Z_t$ observable is the join $\calF_t\vee\sigma(Z_t)$, equivalently splitting exactly the crossed atoms in the finite case. Its value is exact: conditional mutual information under log loss and conditional variance reduction under squared loss, with finite-sample refinement guarded by a sample-split validation bound.

Distinction Graph Machines implement this theorem-to-algorithm path. Nodes store anchored concepts and repairable local memories; edges generate observable distinctions and gate future reads. DGM-Embedding instantiates the same refine-or-repair rule as a detached, forward-only next-token memory layer for language models.

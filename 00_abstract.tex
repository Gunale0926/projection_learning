We introduce Distinction Graph Machines, a runtime-learning model that separates representation refinement from local state repair. The guiding rule is refine when the current concepts are too coarse, and repair when a local concept is adequate but its state is wrong. DGM implements this separation as a graph-generated information structure with anchored concepts, movable retrieval centroids, edge-gated routing, and repairable local memories. Minimal refinement is the smallest partition or sigma-algebra update that makes new evidence measurable, and contradictions inside an atom cannot be solved by any predictor measurable with respect to the current information. The value of a refinement is exact: under log loss it is conditional mutual information, and under squared loss it is conditional variance reduction. The concrete data structure realizes refinement by creating anchors and distinction edges that generate a finite anchor-based information structure; movable centroids are retrieval statistics for the trainable soft read path. We give prediction and observation algorithms with candidate-recall retrieval and guarded repair, sample-split statistically guarded objective-descent guarantees for refine-or-repair, a data-structure complexity theorem, and a language-modeling instance in which hidden-state concepts store next-token memories and produce runtime logit corrections without inference-time backpropagation. The canonical large-model layer is DGM-Embedding, whose detached target is the observed next-token embedding and whose adaptive state is updated by the same sequential forward-only rule used at inference. Adapter-style residuals are treated as non-canonical extensions with explicit target choices. Projection Memory and CPM appear as local repair mechanisms after the relevant distinction structure has been fixed.

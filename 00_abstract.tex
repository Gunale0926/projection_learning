Self-attention stores one key-value entry per previous token and retrieves from
that token table. This gives expressive content-addressed recall, but it ties
the memory basis to token events and makes the read state grow with context
length. We propose \emph{Concept Graph Attention} (CGA), a causal sequence
layer that keeps the continuous readout of attention but replaces token memory
with a fixed bank of runtime concepts. Each concept slot stores an address,
value memory, count statistics, and trainable retention dynamics. Tokens read
from edge-eligible concept slots with attention-style softmax weights, then
write back by gated sufficient-statistic updates. CGA therefore changes where
attention is computed: from token positions to a compact concept bank whose
contents evolve over the prefix.

Theoretically, fixed-bank CGA is an attention-compatible state layer with a
clear scan-friendly update: reads are differentiable concept attention, while
writes are local associative-memory repairs with detached routing and
differentiable value/gate paths. This gives CGA \emph{test-time concept
learning}: at inference, model weights remain fixed, but the concept memory is
learned from the observed prefix by a forward-only update. This separates slow
neural training from fast runtime state mutation without requiring an
inference-time backward pass.
Empirically, we evaluate CGA in the PoST/Zoology sequence-modeling framework on
state-equalized multi-query associative recall and a small causal
language-model setup. The intended comparison class is modern efficient
attention alternatives such as Mamba, RWKV, GDN, and TTT-style stateful layers:
CGA keeps attention-like retrieval, but moves the persistent state from token
memory or learned test-time weights to a bounded, updateable concept memory.

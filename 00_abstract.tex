We study online prediction when runtime memory can either repair payloads at fixed information or refine what future predictions can distinguish. Modeling memory as a finite partition $\Pi$, we prove an algorithm-independent obstruction: every adaptive repair-only learner whose predictions remain $\Pi$-measurable has expected regret at least $T(R_L(\Pi)-R_L(\Pi^\star))$ against any refined oracle $\Pi^\star$, up to refinement cost. Thus missing distinctions cause linear regret independently of the optimizer. We characterize valuable refinements by Bayes-risk decrease. The minimal binary update is the join $\Pi\vee U$; zero value is equivalent to the refined Bayes action being old-measurable. Under log loss the value is $I(Y;U\mid\Pi)$, and under squared loss it is conditional mean-separation energy. We give a sample-split guard accepting refinement only when penalized empirical gain beats the best repair gain by a concentration margin. Guided by the theory, we propose Conceptual Attention (CA): a fixed-bank sequence layer that reads fixed concept identities and causally repairs bounded concept-value memory.

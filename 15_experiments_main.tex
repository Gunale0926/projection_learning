\section{Experiments}
\label{sec:main-experiments}

The main empirical section is organized for two audiences.  First, we report
standard online-adaptation metrics under a shared frozen representation:
prequential prediction, held-out negative log likelihood, accuracy, memory
state size, and ordinary baselines.  Second, we keep controlled diagnostics
that test the paper's mechanism claims: repair-only lower bounds,
consequence-sensitive refinement, and sparse hierarchy recovery.  All online
learners use the same prequential order: predict, observe the consequence, then
update \citep{dawid1984prequential}.  Labels used to define CIFAR hierarchy
diagnostics are never supplied to the non-oracle predictor or update rule.

The experiments should be read as mechanism evidence plus frozen-backbone
adaptation evidence, not as a broad SOTA benchmark.  We nevertheless include
two small scaling pilots at the end of the section because they show where the
method could become competitive after systems work and stronger baselines.

\subsection{ML-facing online adaptation: CIFAR-100 Image-HQS}

The most ML-like experiment is non-oracle CIFAR-100 Image-HQS.  Each round gives
a frozen image embedding $z=\phi(x)$ and a random query mask
$s\in\{0,1\}^{20}$.  The binary consequence is
\[
Y=s_{c(x)}
\]
with label flip probability $\eta=0.02$, where $c(x)$ is the CIFAR-100 coarse
class \citep{krizhevsky2009learning}.  The learner observes the image embedding,
the mask, and the revealed binary consequence, but never observes the fine or
coarse CIFAR label.  The goal is not image classification; it is online
adaptation to a structured binary prediction problem under a frozen visual
representation.

DGM uses image-only routing with a local mask-coordinate posterior memory,
\[
P(Y=1\mid j,s)
\approx
\eta+(1-2\eta)\sum_{c=1}^{20}\pi_{j,c}s_c,
\]
and updates $\pi_j$ only from the binary outcome, the mask, and the soft routing
weights.  The comparison set is deliberately ordinary: a global frequency
model, an online logistic head on $[z,s]$, and an online crossed logistic head
on $[z,s,z\otimes s]$.  Table~\ref{tab:main-results} shows that DGM improves
test NLL over online logistic by $0.0180$ nats with paired bootstrap 95\% CI
$[0.0148,0.0212]$, and over crossed logistic by $0.0085$ nats with CI
$[0.0047,0.0122]$.  Accuracy gains are $2.75$ percentage points over online
logistic and $0.94$ points over crossed logistic.

This result supports local posterior repair under learned visual concepts.  It
does not claim exact recovery of the CIFAR hierarchy graph from images.  Indeed,
the learned image concepts are not coarse-pure enough for a single local split
bit to be the right action; the posterior memory is the appropriate local
repair.  Appendix~\ref{app:image-hqs-audit} gives leakage checks, posterior
ablations, and representation ceilings.

\subsection{Mechanism diagnostic: BalancedMask-HQS}

BalancedMask-HQS isolates graph recovery from visual recognition.  The stream
reveals the true fine label $z$ as the current atom and hides the CIFAR coarse
label $c(z)$.  Each round samples a mask $s\in\{0,1\}^{20}$ with iid
$s_k\sim\operatorname{Bernoulli}(1/2)$ and sets
\[
Y=s_{c(z)}
\]
before flipping with probability $\eta=0.02$.  Since $P(Y=1\mid z)=1/2$, a
repair-only learner that conditions only on $z$ has Bayes log-risk $\log 2$.
The refined oracle that observes the single bit $s_{c(z)}$ has risk
$H_{\rm bin}(\eta)$.  All mask coordinates have identical marginal frequency,
so a frequency-only edge selector has no reliable shortcut; only the
consequence-valued coordinate matters.

The result in Table~\ref{tab:main-results} is intentionally not framed as
"DGM beats every model."  Crossed logistic and a per-fine local stump can fit
the task.  The point is structural: DGM-HQS recovers exactly one
fine-to-coarse edge per fine class, accepts no spurious edges, and reaches the
oracle refined risk with about 200 atoms rather than a full crossed interaction
table.  Budgeted cache baselines, reported in Appendix~\ref{sec:experiments},
store examples but do not recover the reusable mask-coordinate rule under the
reported budgets.

\subsection{Mechanism diagnostic: Bayes-gap calibration}

The Bayes-gap experiment checks the numerical scale of the repair obstruction.
We sample
\[
Z\sim\operatorname{Unif}\{1,\ldots,K\},
\qquad
U\sim\operatorname{Bernoulli}(1/2),
\]
let the old partition observe only $Z$, and let the refined partition observe
$(Z,U)$.  Consequences follow
\[
Y\mid Z=z,U=u
\sim
\operatorname{Bernoulli}(0.5+(-1)^{1-u}\delta).
\]
Thus the repair-only risk is $\log 2$, the refined oracle risk is
$H_{\rm bin}(0.5+\delta)$, and the predicted regret slope is
\[
\Delta_{\rm theory}
=
\log 2-H_{\rm bin}(0.5+\delta)
=
I(Y;U\mid Z).
\]
Figure~\ref{fig:main-evidence}A shows that the estimated repair-only regret
slope matches this conditional mutual information.  At $\delta=0$, the bit has
zero consequence value and the DGM split rule accepts no noise split.  This is
the calibration check for the theorem, not an engineering benchmark.

\subsection{Scaling and SOTA-readiness pilots}

The following pilots are not SOTA claims, but they answer a more concrete
question: is DGM only a diagnostic mechanism, or is there a plausible path to a
competitive ML system without turning the method into ordinary backpropagation?
The current evidence favors the pure route.  Raw pixels and unsupervised PCA are
too weak on CIFAR-10, but a fixed non-neural visual basis plus DGM-style
read/write memory gives a large gain without training a representation.  The
trainable-CNN row is kept only as a non-pure reference for how much headroom
there is when learned features are allowed.

The empirical bar for a future pure-DGM SOTA-ready claim is therefore clear:
run the same prequential contract with matched memory bytes and adaptation time
against strong cache, $k$NN, online ridge/RLS, online tree, prototype-growth,
handcrafted-feature SVM/ridge, and gradient-adapter baselines; report latency
and memory as primary metrics; and show that the gain persists under better
fixed bases and memory-selection policies.  Until that grid is complete, the
main claim remains narrower: DGM implements the refine-or-repair decision and
shows that pure, no-backprop memory can improve sharply once the fixed
distinction basis is not impoverished.

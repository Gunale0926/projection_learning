\section{Experiments}
\label{sec:main-experiments}

The empirical claim is intentionally narrow.
We evaluate CGA as a causal sequence mixer: a layer with the external contract
of attention, but whose memory is a bounded concept graph rather than all
previous tokens.
The main experiment is multi-query associative recall (MQAR), because MQAR
directly measures whether a mixer can write key-value information and retrieve
it later.
Language modeling is the scaling target, but the first publication-level result
should establish controlled recall and state scaling.

\subsection{Experiment 1: Multi-Query Associative Recall}
\label{sec:exp-mqar}

MQAR inputs contain key-value pairs followed by queries; the model must output
the corresponding value \citep{arora2024zoology}.
This is the cleanest test for CGA because the task separates two questions:
whether the model can store associations in a runtime memory, and whether later
queries can retrieve them.
For token attention, the memory is the raw prefix.
For CGA, the memory is a concept graph constructed from the prefix.

We run CGA inside the existing PoST/Zoology MQAR framework.
The comparison is state-equalized against existing efficient sequence mixers.
We use the same grid as the PoST/Mamba benchmark runs:
\[
(d_{\rm model},H)\in\{(512,4),(512,8),(256,4)\},
\qquad
d_{\rm state}=d_{\rm model}/(2H).
\]
These correspond to target runtime-state budgets
$d_{\rm model}^2/H\in\{64{\rm K},32{\rm K},16{\rm K}\}$.
For each grid point, the concept budget $C$ is chosen as the largest integer
whose CGA state size does not exceed the target budget.
The resulting concept budgets are $C=96$ for $(512,4)$, $C=55$ for $(512,8)$,
and $C=47$ for $(256,4)$.
Each grid point is run with the same three-point learning-rate sweep
$\{10^{-3.5},10^{-3},10^{-2.5}\}$.
The implementation uses dense concept reads, untied read/write address
projections, and a learned read logit scale initialized at the standard
dot-product attention scale.
Training uses sequence length $512$ with the same curriculum over key-value
count as the PoST baselines.
Evaluation reports the existing persist and capacity slices at lengths
$512,1024,2048,4096$.

\paragraph{Baselines.}
The comparison class is the standard MQAR suite already present in the lab
framework:
\begin{itemize}
    \item \textbf{Transformer}: token-level causal self-attention.
    \item \textbf{Mamba/RWKV/GDN/GLA/RetNet}: recurrent or linear-state
    efficient mixers.
    \item \textbf{TTT-style layers}: expressive state updated by test-time
    learning style mechanisms.
    \item \textbf{CGA}: attention over a runtime concept graph.
\end{itemize}

\paragraph{What the experiment tests.}
The fair-state run asks whether replacing token memory by concept memory is
competitive at the same state budget.
The important diagnostic slices are:
\[
(\text{test length},\ \text{key-value count},\ \text{persist/capacity}).
\]
A positive result is not merely low training loss.
CGA should maintain recall when the query distance grows, expose a compact
runtime state, and report latency and concept count so that improvements are
not hidden inside unbounded memory.

The implementation artifact is
\path{../PoST_dev/zoology/zoology/experiments/cga_mqar.py}.
The registered mixer is \texttt{zoology.mixers.cga.ConceptGraphAttention}.

\subsection{Experiment 2: CGA-LM}
\label{sec:exp-cga-lm}

The language-model experiment asks whether the same concept-memory read/write
operation can serve as a causal mixer beyond synthetic recall.
The matched comparison is:
\begin{itemize}
    \item \textbf{Tiny Transformer}: standard causal self-attention.
    \item \textbf{TinyCGA-LM}: the attention sublayer is replaced by CGA.
\end{itemize}
Both models should use the same tokenizer, depth, width, optimizer, training
tokens, and evaluation split.
The reported metrics are validation negative log likelihood, perplexity,
prefill latency, decode latency, parameter count, concept count, edge count,
and runtime state size.

The language-model result is not meant to be a large-scale SOTA claim in this
draft.
Its role is to show that CGA is not an MQAR-only mechanism: the same layer that
performs associative recall can be trained as an attention-like decoder mixer.
For larger runs, CGA can be introduced into the existing PoST training stack as
an attention replacement with the same external block interface.

\subsection{Removed Experiments}
\label{sec:exp-not-in-main}

Earlier drafts included CIFAR test-time adaptation, LoRA-style adapters,
entropy minimization, readout-only language-model memory, and synthetic
refinement diagnostics.
Those experiments are removed from the main claim.
They test different questions and obscure the central message:
\[
\text{CGA replaces token attention memory with a dynamic concept graph.}
\]
MQAR tests controlled concept-memory recall.
CGA-LM tests whether the same operation can be trained as a causal
language-model mixer.

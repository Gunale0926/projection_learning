\section*{NeurIPS Paper Checklist}

This file is retained only for the conference-template branch of
\texttt{main.tex}.

\begin{itemize}
\item {\bf Claims.} The abstract and introduction state the paper's scope as a
method proposal: CGA replaces token-indexed attention memory with a fixed bank
of runtime concept slots.
\item {\bf Limitations.} Section~\ref{sec:main-limitations} states the main
limitations, including kernel optimization, retention tuning, concept-count
selection, and the absence of a large-scale SOTA language-model claim.
\item {\bf Theory.} Section~\ref{sec:main-cga} gives the state definition,
read/write equations, state-size accounting, and backpropagation boundary for
fixed-bank CGA.
\item {\bf Experiments.} Section~\ref{sec:main-experiments} centers on MQAR as
the controlled recall benchmark and CGA-LM as the sequence-modeling scaling
target.
\end{itemize}

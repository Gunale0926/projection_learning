\section*{NeurIPS Paper Checklist}

This file is retained only for the conference-template branch of \texttt{main.tex}. The current draft is configured as an arXiv-style theoretical paper via \texttt{\textbackslash def\textbackslash isarxiv\{1\}} in \texttt{\_0\_vars.tex}, so this checklist is not included in the compiled manuscript.

\begin{itemize}
\item {\bf Claims.} The abstract and introduction state the paper's scope as a method proposal with a theoretical diagnosis: runtime excess risk decomposes into a repair gap and a refinement gap, and Distinction Graph Machines estimate these gaps by repairing local concept memories or refining the information structure with concepts and distinction edges.
\item {\bf Limitations.} Appendix~\ref{app:limitations} states the main limitations, including distinction search, finite-sample estimation, and the restricted exact bridge to one-hot partition memories.
\item {\bf Theory assumptions and proofs.} The assumptions appear in the theorem statements in Sections~\ref{sec:preliminaries}, \ref{sec:algorithms}, and \ref{sec:guarantees}; full proofs are in Appendices~\ref{app:refinement-proofs}--\ref{app:algorithm-bridge-proofs}.
\item {\bf Experiments.} The draft includes small controlled experiments for online distinction memory and backpropagation baselines.
\end{itemize}

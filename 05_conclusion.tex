\section{Conclusion}
\label{sec:conclusion}

The paper begins from a simple mismatch. Backpropagation is a powerful primitive for training static differentiable functions, but a runtime learner faces a different demand: it must absorb evidence while preserving what the evidence does not force it to change. Projection Learning formalizes this demand as minimal consistency repair.

The mathematical arc is that projection is characterized by stability, soft projection contains implicit mirror descent, and gradient descent appears as a linearized $\ell_2$ case. Conservative Projection Memory then realizes the principle in a neural associative state. Its central rule is to write along the innovation of a key rather than the key itself. This single change exposes a complete algebra of runtime evidence: exact capacity, redundancy, novelty, contradiction, lower bounds on safe repair, optimal residual contraction, and equivalence between sequential hard repair and batch projection.

The broader conclusion is that runtime learning should make the geometry of evidence visible. A useful adaptive system should be able to say what is already implied, what is genuinely new, what is statistically uncertain, and what cannot be changed without breaking prior commitments. Projection memory supplies one concrete mechanism for doing so with forward computation and closed-form state repair, without inference-time backpropagation.
